Etiquetas nuevas en HTML5

<header>

La etiqueta <header> se utiliza para definir un encabezado para un documento o una seccion. Puede contener elementos como titulos, logotipos y navegacion.

<nav>

La etiqueta <nav> se utiliza para definir un conjunto de enlaces de navegacion. Ayuda a los usuarios a navegar por el sitio web.
